## Hey, I'm Linh.

I currently work as a **Backend Tech Lead** at **FPT Software** in **Da Nang, Vietnam**.

I lead delivery across backend/platform systems, release pipelines, production deployments, and large-scale product work. My background started in Android, expanded into AWS/serverless and backend architecture, and now also includes AI/LLM automation plus a growing OSS toolchain under the **Forge** ecosystem.

- **Enterprise scale:** shipped work for products with **10M+** and **1M+** downloads
- **Backend depth:** Lambda fan-out, SQS DLQ + replay, CDK, realtime systems, TB-scale data movement
- **Builder mode:** publishing developer tooling under `@magicpro97`
- **Current edge:** applying LangGraph, OpenAI, and Gemini to multi-agent automation

## What I'm focused on now

- Leading **backend delivery** with stronger ownership around **deployments, quality gates, and production reliability**
- Building OSS tooling that compresses the path from **idea -> scaffold -> backend -> test -> ship -> monitor**
- Combining **AI workflows** with practical engineering systems instead of demo-only prototypes
